KollectiONX by Brian Millham <brian@millham.net>

KollectiONX is for users of Collectorz.com Movie Collector. KollectiONX
uses the MySQL database created by KollectiON (collectortoolz.co.uk) from
the Movie Collector database.

This allows users of Movie collector to use XBMC on any XBMC supported
platform, and have full access to their Movie Collector database.

Requirements:
A MySQL database created with KollectiON (collectortoolz.co.uk)
XBMC
